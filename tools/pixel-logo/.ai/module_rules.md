# 工具需求

- 生成一个可以制作像素画logo的工具；
- 工具可以像画图那样画logo，只是logo是像素画；
- 可以设置大小，例如64*64和32*32，默认是64*64，用户可以根据需要调整大小；
- 每个像素点支持不同的颜色，有调色盘可以设置；
- 有铅笔，橡皮擦，刷子等绘制方式；
- 支持拖拽导入图片，图片会被转换为像素画，可以设置转换的像素大小；
- 可以输出像素画的代码，包括svg和base64格式；
- 尽量让界面只占一个屏幕；
- 画布上有网格线（各选是否显示），可以设置背景色（包括透明）；
- 颜色设置也包括透明；

# 布局规划

- 整个工具有左中右三个大块区域：
  - 左侧是工具栏，包括尺寸设置，调色盘，绘制方式选择，文件上传拖拽，等等功能设置；
  - 中间是画布，用于绘制像素画；
  - 右侧分为两块：
    - 上面80%区域是输出区域，用于显示输出的像素画代码（如svg和base64）；
    - 下面20%区域显示工具标题、作者、PoweredBy等信息；

# 代码生成规则

- 生成3个文件：pixel-logo.html，pixel-logo.js和pixel-logo.css；
- 界面使用英语，代码内部注释可以用中文；
- 删掉调试用的日志语句；
