# fs-tools

A lightweight JavaScript toolset providing various utilities for development tasks.

## Overview

fs-tools is a collection of JavaScript-based tools designed to assist developers with different development and
visualization needs. Each tool is contained in its own directory within the `tools` folder.

## Features

- **Pure Frontend Code**: All tools are implemented using pure frontend technologies (HTML, CSS, JavaScript)
- Interactive visualization tools
- Responsive design
- Lightweight with minimal dependencies
- Extensible architecture
- Modern, intuitive UI

## Installation

1. Clone the repository
2. Open `index.html` in your web browser

## Available Tools

- **[JMH Visualizer](tools/jmh-visualizer/jmh-visualizer.html)**: A tool for visualizing Java Microbenchmark Harness (
  JMH) performance test results with interactive charts and data analysis features.
- **[Pixel Logo](tools/pixel-logo/pixel-logo.html)**: A tool for creating pixel art logos.
